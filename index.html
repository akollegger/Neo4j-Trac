About Neo4j

Neo4j is a graph database, an enterprise-grade database that is easy to learn, deploy and scale. For the latest details about Neo4j, visit these resources:

  • Neo4j Manual: Comprehensive documentation from development to deployment.
  • Video: Webinars, screencasts and interviews.
  • Google Group: Community discussions and assistance.
  • Meetups: Socialize with other Neo4j users while learning the latest.
  • GitHub Project: Developer-level implementation questions and bug-reports.

Get Neo4j

It's easy to get started with Neo4j. Just download and try it out. Neo4j is available in three editions, each available with commercial licensing:

  • Community: full-powered database
  • Advanced: adds monitoring capability
  • Enterprise: adds live backups and high-availability
